## How I work

- **Guardrails first.** Tests, isolation, HALT switches, `DECISIONS.md` discipline. If a claim ships, a test proves it; if an agent has a permission, a test enforces it stays scoped.
- **Domain depth applied to AI.** I bring the technician's habits — reproduce, isolate, verify — to how I build with LLMs. Every agent I ship is scoped by a charter, gated by explicit quality checks, and paired with a human-in-the-loop model matched to the stakes.
- **Cross-medium output.** Same domain knowledge, multiple output shapes: HTML/`.pptx` decks, `.docx`, VR simulations, native mobile companions. The medium is chosen by what the learner needs, not what the tool makes easy.
